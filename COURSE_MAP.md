阶段五的库学习连接：

- `numpy`：把文本向量、相似度、矩阵计算先变成可理解的数组问题。
- `sklearn.metrics`：把模型输出变成可评估的分数和错误案例。
- `torch`：理解 Tensor、Embedding、Linear、loss 和 optimizer 的最小概念。
- `transformers`：理解 tokenizer、model、pipeline 的输入输出，不要求训练模型。
- `pydantic`：把 LLM 输出和 Agent tool 参数变成可校验结构。
