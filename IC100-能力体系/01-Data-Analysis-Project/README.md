# Data Analysis Project - LLM Evaluation Analytics Platform

这个项目替代原来的泛化电商数据分析。目标是做一个适合 AI / LLM / Agent 求职方向的数据分析作品：把 LLM 评估日志转化为可清洗、可查询、可视化、可复现、可展示的分析平台。

## 最终作品

- 项目名：LLM Evaluation Analytics Platform。
- 技术栈：Python、JSONL/CSV、pandas、NumPy、SQLite、SQL、Matplotlib、Streamlit。
- 产出：clean data、SQLite warehouse、metrics tables、figures、Streamlit dashboard、final report、README、resume bullets。

## 为什么这个项目更适合你

它连接统计学、Python 数据分析和 LLM 应用真实工作流。你不是只做图表，而是在训练“评估一个 AI 系统是否可靠”的数据能力：字段设计、指标口径、统计解释、错误案例、成本延迟和可复现交付。
