# IC100 能力体系

IC100 是 Python100 的能力验证线。

```text
Python100 = 知识体系
IC100 = 能力体系
GitHub = 成果体系
LeetCode = 算法体系
```

目标人群：统计学硕士 -> AI Native Builder -> Agent / LLM / Quant 方向实习求职。

核心原则：保留现有 Python100 内容，不推翻重做；Day 只负责 What / Why / How / Common Errors / Future Usage；IC 只负责能力验证、工程训练和作品沉淀。

## 结构

- [LeetCode 200 题](LEETCODE_200.md)：每天 2 题，官方链接，不重复。
- IC001-IC019：基础能力启动，连接 Day01-Day19。
- IC020-IC039：Data Analysis Project，最终形成电商用户行为分析。
- IC040-IC059：Quant Research Project，最终形成 Mini Quant Research Framework。
- IC060-IC079：LLM Application Project，最终形成 RAG Knowledge Assistant。
- IC080-IC100：Agent Workflow Project，最终形成 Agent Workflow System。
