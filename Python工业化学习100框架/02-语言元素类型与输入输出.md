## 知识点 1：变量 Variable

### 定义

变量是给值起名字，方便后面复用。

### 为什么存在

它让当前阶段的代码更容易看懂、运行和复查。
## 知识点 2：类型 Type

### 定义

类型决定一个值能不能计算、拼接或格式化。

### 为什么存在

它让当前阶段的代码更容易看懂、运行和复查。

### 5 道基础巩固题

#### 1. 基础巩固 - 学习卡片变量版

题目：用变量保存学习主题、时间、目标，并打印。

讲解：今天开始减少硬编码。

```python
topic = "Python"
hours = 2
goal = "能理解变量"
print(f"主题：{topic}")
print(f"时间：{hours} 小时")
print(f"目标：{goal}")
```

#### 2. 基础巩固 - 金额计算

题目：用价格和数量计算金额。

讲解：只用变量和乘法。

```python
price = 99.0
quantity = 3
amount = price * quantity
print(f"金额：{amount}")
```

#### 3. 基础巩固 - 输入后输出

题目：读取名字并打印欢迎语。

讲解：`input()` 得到的是字符串。

```python
name = input("请输入代号：")
print(f"欢迎，{name}")
```

#### 4. 基础巩固 - 字符串数字转换

题目：把输入的小时数转成整数。

讲解：转换后才能做加法。

```python
raw_hours = input("学习小时数：")
hours = int(raw_hours)
print(f"明天可以学 {hours + 1} 小时")
```

#### 5. 基础巩固 - 收益率最小计算

题目：用两个价格变量计算收益率。

讲解：不写函数，先理解公式。

```python
start_price = 100.0
end_price = 108.0
return_rate = (end_price - start_price) / start_price
print(f"收益率：{return_rate:.2%}")
```
