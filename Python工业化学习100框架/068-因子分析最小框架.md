### 带注释参考代码：Subarray Sum Equals K

```python
def subarray_sum(nums, k):
    prefix = 0
    counts = {0: 1}
    total = 0
    for num in nums:
        prefix += num
        total += counts.get(prefix - k, 0)
        counts[prefix] = counts.get(prefix, 0) + 1
    return total
```

## 强化练习与工程使用

> 所属阶段：Quant 研究与回测基础。目标是：把行情样例变成可复查的研究实验。

### 工程使用场景

今天的主题是 **因子分析最小框架**，不要只把它当成一个孤立知识点。它应该落到 `Quant Research Starter` 里的一个可复用能力：输入清楚、处理过程可检查、输出可以保存，失败时能解释原因。重点检查：未来函数、样本区间、交易成本、风险指标。

### 强化 7 题

1. **复现题**：不看答案重写今天最小案例，并写出运行命令和预期输出。
