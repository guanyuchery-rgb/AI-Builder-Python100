## 今日最小可运行任务

```python
def validate_report(data: dict) -> list[str]:
    errors = []
    for field in ["summary", "risk", "next_step"]:
        if field not in data:
            errors.append(f"missing {field}")
    return errors

print(validate_report({"summary": "ok", "risk": "low"}))
```
