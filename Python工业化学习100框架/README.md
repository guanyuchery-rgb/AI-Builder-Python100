# Python 工业化学习 100 天

这是一个面向 AI Builder、数据分析、Quant、LLM 和 Agent 项目的开源 Python 100 天学习路径。

课程设计原则：零基础起步、难度逐步承接、每天都有可运行代码、每个阶段都能沉淀成公开项目资产。

## 使用方式

- 每天按 2 小时节奏学习。
- 先跑通最小案例，再做 7 道简单路线题和 5 道基础巩固题。
- Day01-Day20 不安排 Hot100 难题；Day21 之后每天追加 1 道 Hot100 / LeetCode 挑战，完成主线后再做。
- 所有示例使用虚拟数据，不提交真实身份、密钥或私人资料。
- 学完一个阶段，至少整理一次 README 或项目复盘。
