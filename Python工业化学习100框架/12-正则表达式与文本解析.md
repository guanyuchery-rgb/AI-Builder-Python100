### 5 道基础巩固题

#### 1. 基础巩固 - 提取数字

题目：从 price 文本提取数字。

讲解：先用 search。

```python
import re

text = "price=123.45"
match = re.search(r"\d+\.\d+", text)
if match:
    print(match.group())
```

#### 2. 基础巩固 - 日期形状

题目：判断 YYYY-MM-DD。

讲解：只判断形状，不判断真实日期。

```python
import re

date = "2026-06-01"
print(re.fullmatch(r"\d{4}-\d{2}-\d{2}", date) is not None)
```

#### 3. 基础巩固 - 提取邮箱

题目：从文本找邮箱。

讲解：findall 返回列表。

```python
import re

text = "contact: demo@example.com"
print(re.findall(r"[\w.-]+@[\w.-]+\.\w+", text))
```

#### 4. 基础巩固 - 提取 ticker

题目：提取 $AAPL。

讲解：量化文本常见。

```python
import re

text = "watch $AAPL and $MSFT"
print(re.findall(r"\$[A-Z]{1,5}", text))
```

#### 5. 基础巩固 - 解析日志

题目：提取 ERROR 后面的内容。

讲解：为 Debug 做准备。

```python
import re

line = "ERROR timeout"
match = re.search(r"ERROR (.*)", line)
if match:
    print(match.group(1))
```
