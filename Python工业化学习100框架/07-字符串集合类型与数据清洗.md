## 知识点 1：字符串清洗

### 定义

字符串是文本数据。字符串方法可以转换、查找、切分、拼接和规范化文本。

### 为什么存在

真实数据经常不干净：多余空格、大小写不一致、分隔符混乱、缺失值等。

### 最小案例

```python
raw_symbol = "  aapl "
symbol = raw_symbol.strip().upper()
print(symbol)
```

### 常见错误

- 忘记字符串不可变。
- 混淆 `split()` 和 `strip()`。
- 用脆弱切片处理结构化文本。

### 工程应用

- 清洗 CSV 字段。
- 统一股票代码。
- 解析日志。
- 准备 prompt 输入。

### 未来扩展

- 正则表达式。
- 文本编码。
- NLP 预处理。

## 知识点 2：常用集合类型

### 定义

Python 常用集合类型：

- `list`：有序、可修改。
- `dict`：键值映射。
- `set`：去重集合。
- `tuple`：固定有序值。

### 为什么存在

不同数据形状需要不同容器。容器选对了，代码会更简单、更安全。

### 最小案例

```python
prices = [100, 102, 101]
stock = {"symbol": "AAPL", "price": 102.0}
unique_symbols = {"AAPL", "MSFT", "AAPL"}
point = ("2026-05-31", 102.0)

print(prices[0])
print(stock["symbol"])
print(unique_symbols)
print(point)
```

### 常见错误

- `KeyError`：字典里没有这个键。
- `IndexError`：列表下标超范围。
- 期待 set 保持顺序。
- 遍历 list 时同时修改它。

### 工程应用

- API 返回记录。
- 配置字典。
- 唯一 ID 检查。
- DataFrame 之前的原始行数据。

### 未来扩展

- `collections.Counter`。
- `defaultdict`。
- JSON。
- pandas DataFrame。

## 现代代码示例：清洗交易记录

```python
raw_rows = [
    " aapl , 100.5 , buy ",
    " MSFT,  210.0,sell",
]

records = []
for row in raw_rows:
    symbol, price, side = [part.strip() for part in row.split(",")]
    records.append({
        "symbol": symbol.upper(),
        "price": float(price),
        "side": side.lower(),
    })

print(records)
```

### 5 道递进题

#### 1. Easy - Valid Palindrome

关联题：[LeetCode 125](https://leetcode.com/problems/valid-palindrome/)

题目：忽略大小写和非字母数字，判断是否回文。

讲解：先清洗再判断，比边遍历边处理更容易读。

```python
# isalnum 保留字母和数字，lower 统一大小写
def is_palindrome(s: str) -> bool:
    cleaned = "".join(ch.lower() for ch in s if ch.isalnum())
    return cleaned == cleaned[::-1]
```

#### 2. Easy - First Unique Character

关联题：[LeetCode 387](https://leetcode.com/problems/first-unique-character-in-a-string/)

题目：返回第一个只出现一次的字符下标。

讲解：先统计，再按原顺序找答案。

```python
from collections import Counter

def first_uniq_char(s: str) -> int:
    counts = Counter(s)
    for i, ch in enumerate(s):
        if counts[ch] == 1:
            return i
    return -1
```

#### 3. Medium - Group Anagrams

关联题：[LeetCode 49](https://leetcode.com/problems/group-anagrams/)

题目：把异位词分到同一组。

讲解：排序后的字符串可以作为统一 key。

```python
from collections import defaultdict

def group_anagrams(words: list[str]) -> list[list[str]]:
    groups = defaultdict(list)
    for word in words:
        key = "".join(sorted(word))
        groups[key].append(word)
    return list(groups.values())
```

#### 4. Medium - CSV 行清洗

题目：清洗一行用户数据，去空格、统一邮箱小写。

讲解：真实数据工程大量时间花在这种小清洗上。

```python
# 输入: dict 原始行
# 输出: dict 标准行
def clean_user_row(row: dict[str, str]) -> dict[str, str]:
    return {
        "name": row.get("name", "").strip(),
        "email": row.get("email", "").strip().lower(),
        "city": row.get("city", "").strip(),
    }
```

#### 5. Hard - Prompt 消息去重

题目：给消息列表，去掉完全重复的 `role + content`。

讲解：LLM 应用里重复上下文会浪费 token，也会污染结果。

```python
# 输入: [{"role": "user", "content": "..."}]
# 输出: 保留原顺序的去重结果
def dedupe_messages(messages: list[dict[str, str]]) -> list[dict[str, str]]:
    seen = set()
    result = []
    for msg in messages:
        key = (msg.get("role"), msg.get("content"))
        if key not in seen:
            seen.add(key)
            result.append(msg)
    return result
```
