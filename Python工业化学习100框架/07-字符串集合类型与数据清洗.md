### 5 道基础巩固题

#### 1. 基础巩固 - 字符串清洗

题目：清洗一个 symbol。

讲解：字符串方法是数据清洗第一步。

```python
symbol = " aapl "
clean = symbol.strip().upper()
print(clean)
```

#### 2. 基础巩固 - 字典读取

题目：用 dict 保存一条价格记录。

讲解：字段名比位置更清楚。

```python
row = {"symbol": "AAPL", "price": 100.5}
print(row["symbol"])
print(row.get("volume", 0))
```

#### 3. 基础巩固 - 清洗函数

题目：写函数清洗 symbol。

讲解：函数 + 字符串。

```python
def clean_symbol(symbol):
    return symbol.strip().upper()

print(clean_symbol(" msft "))
```

#### 4. 基础巩固 - 价格字段转换

题目：把 dict 里的价格字符串转成 float。

讲解：只处理存在字段。

```python
row = {"price": "101.2"}
price = float(row.get("price", 0))
print(price)
```

#### 5. 基础巩固 - 标签去重

题目：用 set 去重标签。

讲解：set 只保留唯一值。

```python
tags = ["python", "sql", "python"]
unique_tags = set(tags)
print(unique_tags)
```
