## 基础详细讲解

### 分支结构与输入验证

`if / elif / else` 是让程序根据条件走不同路径。你可以把它想成一张规则表：先检查最特殊、最危险、最需要拦截的情况，再处理正常情况。条件表达式的结果只有 True 或 False。

### 手写顺序

- 每个 `if` 后面必须有英文冒号，下面缩进的代码才属于这个分支。
- 先写非法输入检查，例如空值、负数、超出范围，再写正常业务逻辑。
- `elif` 用来表达互斥条件，`else` 负责兜底；不要把所有判断都写成独立的 `if`。

### 今天必须真正弄懂

- **输入**：今天这段代码从哪里拿到数据，是手写值、终端输入、文件，还是函数参数。
- **处理**：每一行代码改变了什么状态，例如变量变了、列表多了一个元素、判断走了某个分支。
- **输出**：结果给谁看，是终端输出、返回值、文件，还是后面函数继续使用。

### 和后续知识的关系

Agent tool、API 和数据清洗都需要输入验证。今天练的是“先挡住坏输入，再计算好结果”。

### 零基础拆解：条件判断和输入验证

#### 这一天到底在练什么

分支结构处理的是规则。你输入一个值，程序根据条件决定走哪条路。初学先把条件写成简单中文：如果小于 0、如果等于 0、否则。

#### 手写时按这个顺序

- 先写一个只包含 `if` 的判断。
- 再加入 `elif` 处理第二种情况。
- 最后加入 `else` 兜底，并确认每个分支只会走一个。

#### 最常见卡点

冒号和缩进是分支最常见的卡点。看到 `IndentationError` 时，先看 if 下面是否缩进了 4 个空格。

## 5 道基础巩固题

#### 1. 合法范围判断

题目：判断分数是否在 0 到 100 之间。

讲解：先处理非法输入。

```python
score = 105
if score < 0 or score > 100:
    print("非法")
else:
    print("合法")
```

#### 2. 学习状态判断

题目：完成度大于等于 80 输出稳定。

讲解：练习 if/else。

```python
progress = 85
if progress >= 80:
    print("稳定")
else:
    print("需要复盘")
```

#### 3. 三档分类

题目：按高、中、低输出状态。

讲解：练习 elif。

```python
score = 72
if score >= 90:
    print("A")
elif score >= 60:
    print("B")
else:
    print("C")
```

#### 4. 正负收益判断

题目：判断收益率是正、负还是零。

讲解：Quant 最小判断。

```python
ret = -0.01
if ret > 0:
    print("positive")
elif ret < 0:
    print("negative")
else:
    print("zero")
```

#### 5. 空文本判断

题目：判断输入是否为空。

讲解：为后面数据清洗做准备。

```python
text = ""
if text == "":
    print("empty")
else:
    print(text)
```
