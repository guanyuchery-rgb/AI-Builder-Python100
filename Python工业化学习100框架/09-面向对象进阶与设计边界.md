### 5 道基础巩固题

#### 1. 基础巩固 - 计算类

题目：写一个只算金额的类。

讲解：先分清职责。

```python
class ValueCalculator:
    def value(self, price, shares):
        return price * shares

calc = ValueCalculator()
print(calc.value(100, 3))
```

#### 2. 基础巩固 - 记录仓库

题目：写一个内存列表仓库。

讲解：仓库只负责保存。

```python
class MemoryStore:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

store = MemoryStore()
store.add("Python")
print(store.items)
```

#### 3. 基础巩固 - 职责说明

题目：给类写注释说明负责什么。

讲解：设计边界先用文字表达。

```python
class LearningStore:
    # 只负责保存学习记录，不负责计算分数
    def __init__(self):
        self.logs = []
```

#### 4. 基础巩固 - 组合使用

题目：用 Store 保存，用 Calculator 计算。

讲解：两个简单对象协作。

```python
store = MemoryStore()
store.add(100)
store.add(200)
calc = ValueCalculator()
print(calc.value(store.items[0], 2))
```
