## 知识点 1：property

### 定义

`property` 可以把方法暴露得像属性一样，常用于计算值或受控访问。

### 为什么存在

它让对象接口更干净，同时把计算或校验逻辑放在内部。

### 最小案例

```python
from dataclasses import dataclass


@dataclass
class Position:
    symbol: str
    shares: int
    price: float

    @property
    def market_value(self) -> float:
        return self.shares * self.price


position = Position("NVDA", 2, 900.0)
print(position.market_value)
```

### 常见错误

- 在 property 里做很重的计算。
- property 访问时产生副作用。
- 修改规则不清楚。

### 工程应用

- 计算指标。
- 派生字段。
- 清晰模型接口。

### 未来扩展

- setter。
- cached property。
- Pydantic computed field。

## 知识点 2：继承与组合

### 定义

继承表达“是一个”的关系。组合表达“拥有一个”的关系。

### 为什么存在

二者都能组织代码复用，但组合通常更容易修改和测试。

### 最小案例

```python
from dataclasses import dataclass


@dataclass
class DataSource:
    name: str

    def fetch(self) -> list[float]:
        return [100, 101, 103]


@dataclass
class ReturnAnalyzer:
    source: DataSource

    def latest_return(self) -> float:
        prices = self.source.fetch()
        return (prices[-1] - prices[-2]) / prices[-2]


analyzer = ReturnAnalyzer(DataSource("demo"))
print(f"{analyzer.latest_return():.2%}")
```

### 常见错误

- 太早设计深继承树。
- 为了复用代码强行继承。
- 一个类同时做数据获取、计算和展示。

### 工程应用

- 策略使用数据源。
- Agent 使用工具。
- 服务使用仓库。
- 分析器使用计算器。

### 未来扩展

- Protocol。
- 依赖注入。
- 抽象基类。

## 现代代码示例：classmethod 构造对象

```python
from dataclasses import dataclass


@dataclass
class Trade:
    symbol: str
    price: float
    quantity: int

    @classmethod
    def from_csv_row(cls, row: str) -> "Trade":
        symbol, price, quantity = [part.strip() for part in row.split(",")]
        return cls(symbol=symbol.upper(), price=float(price), quantity=int(quantity))


trade = Trade.from_csv_row(" aapl, 102.5, 10 ")
print(trade)
```

### 5 道递进题

#### 1. Easy - Min Stack

关联题：[LeetCode 155](https://leetcode.com/problems/min-stack/)

题目：栈支持 `push/pop/top/getMin`。

讲解：额外维护一个最小值栈，避免每次重新扫描。

```python
class MinStack:
    def __init__(self):
        self.stack = []
        self.min_stack = []

    def push(self, val: int) -> None:
        self.stack.append(val)
        if not self.min_stack or val <= self.min_stack[-1]:
            self.min_stack.append(val)

    def pop(self) -> None:
        val = self.stack.pop()
        if val == self.min_stack[-1]:
            self.min_stack.pop()

    def top(self) -> int:
        return self.stack[-1]

    def getMin(self) -> int:
        return self.min_stack[-1]
```

#### 2. Medium - Browser History

关联题：[LeetCode 1472](https://leetcode.com/problems/design-browser-history/)

题目：实现浏览器前进后退。

讲解：状态边界要明确：访问新页面会清掉 forward 历史。

```python
class BrowserHistory:
    def __init__(self, homepage: str):
        self.history = [homepage]
        self.index = 0

    def visit(self, url: str) -> None:
        self.history = self.history[: self.index + 1]
        self.history.append(url)
        self.index += 1

    def back(self, steps: int) -> str:
        self.index = max(0, self.index - steps)
        return self.history[self.index]

    def forward(self, steps: int) -> str:
        self.index = min(len(self.history) - 1, self.index + steps)
        return self.history[self.index]
```

#### 3. Medium - Repository 边界

题目：写一个内存版 `QuoteRepository`，支持 save/list。

讲解：Repository 把数据存储细节和业务逻辑分开。

```python
class QuoteRepository:
    def __init__(self):
        self.rows: list[dict] = []

    def save(self, quote: dict) -> None:
        self.rows.append(quote)

    def list_by_symbol(self, symbol: str) -> list[dict]:
        return [row for row in self.rows if row.get("symbol") == symbol]
```

#### 4. Medium - 错误边界

题目：服务函数不直接读写文件，只接收 repository。

讲解：这样未来换 CSV、SQLite、API 时核心逻辑不用大改。

```python
# repo 只要实现 list_by_symbol，就可以被替换
def latest_price(repo: QuoteRepository, symbol: str) -> float | None:
    rows = repo.list_by_symbol(symbol)
    if not rows:
        return None
    return rows[-1].get("price")
```

#### 5. Hard - LRU Cache

关联题：[LeetCode 146](https://leetcode.com/problems/lru-cache/)

题目：容量满时淘汰最久未使用的 key。

讲解：先用 OrderedDict 版本理解业务语义。

```python
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.data = OrderedDict()

    def get(self, key: int) -> int:
        if key not in self.data:
            return -1
        self.data.move_to_end(key)
        return self.data[key]

    def put(self, key: int, value: int) -> None:
        if key in self.data:
            self.data.move_to_end(key)
        self.data[key] = value
        if len(self.data) > self.capacity:
            self.data.popitem(last=False)
```
