### 带注释参考代码：Best Time to Buy and Sell Stock

```python
def max_profit(prices):
    min_price = float("inf")
    best = 0
    for price in prices:
        min_price = min(min_price, price)
        best = max(best, price - min_price)
    return best
```

## 强化练习与工程使用

> 所属阶段：数据分析与本地数据系统。目标是：把原始数据变成可检查的数据资产。

### 工程使用场景

今天的主题是 **NumPy 数组与向量化计算**，不要只把它当成一个孤立知识点。它应该落到 `本地指标管道` 里的一个可复用能力：输入清楚、处理过程可检查、输出可以保存，失败时能解释原因。重点检查：字段含义、样本数量、缺失值、重复值、时间边界。

### 强化 7 题

1. **复现题**：不看答案重写今天最小案例，并写出运行命令和预期输出。
2. **边界题**：构造 3 个异常输入，说明程序应该返回结果、跳过、还是报错。
3. **数据题**：准备 5-10 条虚拟样例数据，覆盖正常值、空值、重复值和极端值。
4. **函数题**：把核心逻辑封装成 1 个函数，函数名、参数名和返回值必须能读懂。
5. **测试题**：写 2 个正常测试和 1 个边界测试，确认修改后行为不变。
6. **算法题**：完成 [Top K Frequent Elements](https://leetcode.com/problems/top-k-frequent-elements/)，题型：堆 / TopK。提示：先计数再取 TopK，连接日志分析和检索排序。

### 难度控制

Easy 到 Medium，重点是数组、哈希表、字符串和窗口，先追求写清楚。 把每一道题都落到数据检查：行数、字段、空值、重复、分组结果。 如果算法题暂时写不出完整代码，先写暴力思路、输入输出样例和复杂度，再补优化版。
