## 知识点结构

### 定义

用 if/elif/else 做最小判断，是把一个具体问题拆成可运行、可检查、可复用代码的过程。

### 为什么存在

它让学习从临时笔记变成长期资产。公开仓库里的每一天都应该能让读者看到：今天学什么、为什么学、怎么运行、如何检查结果。

### 最小案例

```python
score = float(input("请输入完成度 0-100："))
if score < 0 or score > 100:
    print("输入不合法")
elif score >= 80:
    print("状态：稳定推进")
else:
    print("状态：需要复盘")
```

### 常见错误

- 提前使用循环、函数或复杂异常处理。
- 分支条件覆盖不完整，导致某些输入没有输出。
- 把 `=` 当成比较，比较应该使用 `==`。

## 5 道基础巩固题

#### 1. 合法范围判断

题目：判断分数是否在 0 到 100 之间。

讲解：先处理非法输入。

```python
score = 105
if score < 0 or score > 100:
    print("非法")
else:
    print("合法")
```

#### 2. 学习状态判断

题目：完成度大于等于 80 输出稳定。

讲解：练习 if/else。

```python
progress = 85
if progress >= 80:
    print("稳定")
else:
    print("需要复盘")
```

#### 3. 三档分类

题目：按高、中、低输出状态。

讲解：练习 elif。

```python
score = 72
if score >= 90:
    print("A")
elif score >= 60:
    print("B")
else:
    print("C")
```

#### 4. 正负收益判断

题目：判断收益率是正、负还是零。

讲解：Quant 最小判断。

```python
ret = -0.01
if ret > 0:
    print("positive")
elif ret < 0:
    print("negative")
else:
    print("zero")
```

#### 5. 空文本判断

题目：判断输入是否为空。

讲解：为后面数据清洗做准备。

```python
text = ""
if text == "":
    print("empty")
else:
    print(text)
```
