## 知识点 1：分支 Branch

### 定义

程序根据条件选择不同路径。

### 为什么存在

它让当前阶段的代码更容易看懂、运行和复查。
## 知识点 2：输入验证 Validation

### 定义

先判断输入是否合理，再继续计算。

### 为什么存在

它让当前阶段的代码更容易看懂、运行和复查。

### 5 道基础巩固题

#### 1. 基础巩固 - 及格判断

题目：输入分数，打印是否及格。

讲解：只用一个 if/else。

```python
score = int(input("分数："))
if score >= 60:
    print("及格")
else:
    print("不及格")
```

#### 2. 基础巩固 - 价格验证

题目：输入价格，价格必须大于 0。

讲解：坏输入要先拦住。

```python
price = float(input("价格："))
if price > 0:
    print("价格有效")
else:
    print("价格必须大于 0")
```

#### 3. 基础巩固 - 方向判断

题目：输入交易方向，判断 BUY/SELL/HOLD。

讲解：用 elif 表达多分支。

```python
side = input("方向：").strip().upper()
if side == "BUY":
    print("买入")
elif side == "SELL":
    print("卖出")
else:
    print("观望")
```

#### 4. 基础巩固 - 风险等级

题目：根据亏损比例判断风险。

讲解：只用数字比较，不引入函数。

```python
loss = float(input("亏损比例，例如 0.08："))
if loss >= 0.2:
    print("高风险")
elif loss >= 0.1:
    print("中风险")
else:
    print("低风险")
```

#### 5. 基础巩固 - 空输入检查

题目：输入主题为空时提示重填。

讲解：真实脚本最常见的是输入为空。

```python
topic = input("学习主题：").strip()
if topic == "":
    print("主题不能为空")
else:
    print(f"今天学习：{topic}")
```
