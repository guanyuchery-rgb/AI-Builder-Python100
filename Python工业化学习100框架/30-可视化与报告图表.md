```python
import pandas as pd

summary = pd.DataFrame({"symbol": ["A", "B"], "avg_return": [0.01, -0.005]})
ax = summary.plot(kind="bar", x="symbol", y="avg_return", title="Average Return")
fig = ax.get_figure()
fig.savefig("avg_return.png", dpi=150, bbox_inches="tight")
Path_text = "# Report

已生成 avg_return.png
"
open("report.md", "w", encoding="utf-8").write(Path_text)
```
