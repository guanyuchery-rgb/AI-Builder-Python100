## 知识点 1：Prompt Template

### 定义

Prompt 模板是可复用的指令结构。

### 为什么存在

它让 LLM 输出更稳定、更容易评估。

### 最小案例

- 直接复制复杂代码，但解释不清每一步。
- 跑通后没有保存结果或记录失败情况。

## 今日最小可运行任务

```python
def build_prompt(topic: str, facts: list[str]) -> str:
    fact_text = "
".join(f"- {item}" for item in facts)
    return f"请只基于以下事实分析 {topic}:
{fact_text}
输出：结论、依据、风险。"

print(build_prompt("收益报告", ["样本 20 行", "平均收益 1%"] ))
```
