### 5 道基础巩固题

#### 1. 基础巩固 - 单个处理

题目：写一个大写转换函数。

讲解：先会单任务。

```python
def process_item(item):
    return item.strip().upper()

print(process_item(" aapl "))
```

#### 2. 基础巩固 - 顺序批量

题目：顺序处理多个 symbol。

讲解：先不用线程。

```python
symbols = [" aapl ", " msft "]
for symbol in symbols:
    print(process_item(symbol))
```

#### 3. 基础巩固 - 耗时记录

题目：记录任务耗时。

讲解：以后再比较并发。

```python
import time

start = time.time()
time.sleep(1)
end = time.time()
print(f"耗时：{end - start:.2f}s")
```

#### 4. 基础巩固 - 结果列表

题目：把处理结果保存到列表。

讲解：复用 append。

```python
symbols = [" aapl ", " msft "]
clean = []
for symbol in symbols:
    clean.append(process_item(symbol))
print(clean)
```

#### 5. 基础巩固 - 并发概念记录
