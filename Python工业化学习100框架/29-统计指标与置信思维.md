## 今日最小可运行任务

```python
import math

returns = [0.01, -0.02, 0.03, 0.00, 0.015]
mean = sum(returns) / len(returns)
variance = sum((x - mean) ** 2 for x in returns) / (len(returns) - 1)
std = math.sqrt(variance)
se = std / math.sqrt(len(returns))
print({"mean": mean, "std": std, "se": se})
```
