## 基础详细讲解

### 循环、range、累计器与计数器

`for` 解决重复任务。`range(101)` 生成 0 到 100，循环每次拿到一个数。`total += x` 是把本次结果累计进总结果，`count += 1` 是记录发生次数。它们是最早的算法思维。

### 手写顺序

- 先写清楚要遍历什么：数字范围、列表、字符串，还是文件里的每一行。
- 循环前初始化结果，例如 `total = 0`；循环内只做一小步更新。
- 循环结束后再输出总结果，不要在循环里把最终结论打印很多次。

### 今天必须真正弄懂

- **输入**：今天这段代码从哪里拿到数据，是手写值、终端输入、文件，还是函数参数。
- **处理**：每一行代码改变了什么状态，例如变量变了、列表多了一个元素、判断走了某个分支。
- **输出**：结果给谁看，是终端输出、返回值、文件，还是后面函数继续使用。

### 和后续知识的关系

收益累计、上涨天数统计、日志行数统计、批量处理文件，本质上都来自今天这两个模式。

### 零基础拆解：循环、累计器和计数器

#### 这一天到底在练什么

循环就是“同一套动作做多次”。累计器保存总量，计数器保存次数。你以后算总收益、上涨天数、交易次数，都会回到这两个模式。

#### 手写时按这个顺序

- 先写 `total = 0`，明确总结果从 0 开始。
- 再写 `for x in range(1, 6):`，一次只处理一个 x。
- 循环里面写 `total += x`，循环外面打印 total。

#### 最常见卡点

如果结果被打印很多次，通常是 `print(total)` 缩进到了循环里面；如果少算一个数，检查 `range()` 的结束值是否不包含。

## 5 道基础巩固题

#### 1. 打印 Day 编号

题目：输出 Day1 到 Day5。

讲解：练习 range。

```python
for day in range(1, 6):
    print(f"Day{day}")
```

#### 2. 累计学习小时

题目：每天 2 小时，累计 7 天。

讲解：练习累计变量。

```python
total = 0
for day in range(1, 8):
    total = total + 2
print(total)
```

#### 3. 计算 1 到 100 求和

题目：用循环累计 1 到 100。

讲解：算法基本功。

```python
total = 0
for n in range(1, 101):
    total = total + n
print(total)
```

#### 4. 复利演示

题目：本金每天增长 1%。

讲解：Quant 直觉练习。

```python
money = 100
for day in range(1, 6):
    money = money * 1.01
    print(money)
```

#### 5. 倒计时

题目：从 5 输出到 1。

讲解：理解 range 步长。

```python
for n in range(5, 0, -1):
    print(n)
```
