## 知识点 1：脚本 Script

### 定义

`.py` 文件就是脚本，解释器会从上到下执行。

### 为什么存在

它让当前阶段的代码更容易看懂、运行和复查。
## 知识点 2：输出 Output

### 定义

`print()` 用来把结果显示出来，是最早的反馈方式。

### 为什么存在

它让当前阶段的代码更容易看懂、运行和复查。

### 5 道基础巩固题

#### 1. 基础巩固 - 打印三行文字

题目：连续打印学习方向、今天目标、完成标准。

讲解：只练习 `print()`，不要用变量。

```python
print("学习方向：Python 基础")
print("今天目标：能运行脚本")
print("完成标准：看到终端输出")
```

#### 2. 基础巩固 - 打印简单计算

题目：打印加法、减法、乘法结果。

讲解：先确认 Python 能做基本计算。

```python
print(10 + 5)
print(10 - 5)
print(10 * 5)
```

#### 3. 基础巩固 - 区分文字和数字

题目：分别打印 `"2 + 3"` 和 `2 + 3`。

讲解：理解引号里的内容只是文字。

```python
print("2 + 3")
print(2 + 3)
```

#### 4. 基础巩固 - 写可读输出

题目：输出时加文字说明。

讲解：结果要让未来自己看得懂。

```python
print("今日学习分钟数：")
print(120)
```

#### 5. 基础巩固 - 最小 Debug 记录

题目：写一段会报错的代码，再写修复版。

讲解：第一天只认识错误，不要求完全理解。

```python
# 错误示例：print("hello)
# 修复示例：
print("hello")
```
