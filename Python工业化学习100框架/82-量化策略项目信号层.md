### 5 道递进题

#### 1. Easy - 收益率函数

关联题：[LeetCode 121](https://leetcode.com/problems/best-time-to-buy-and-sell-stock/)

题目：根据起始价格和结束价格计算收益率。

讲解：收益率口径先统一，后面所有指标才可信。

```python
def simple_return(start: float, end: float) -> float:
    if start <= 0:
        raise ValueError("start must be positive")
    return end / start - 1
```

#### 2. Easy - 批量收益率

题目：把价格序列转成收益率序列。

讲解：时间顺序不能乱。

```python
def returns_from_prices(prices: list[float]) -> list[float]:
    return [simple_return(prices[i - 1], prices[i]) for i in range(1, len(prices))]
```

#### 3. Medium - 最大回撤

题目：计算净值序列最大回撤。

讲解：最大回撤是历史高点后的最大下跌。

```python
def max_drawdown(equity: list[float]) -> float:
    peak = equity[0]
    worst = 0.0
    for value in equity:
        peak = max(peak, value)
        worst = min(worst, value / peak - 1)
    return worst
```

#### 4. Medium - 策略指标包

题目：输出平均收益、波动率、最大回撤。

讲解：单个收益数字不够，要一起看风险。

```python
import numpy as np

def strategy_metrics(returns: list[float], equity: list[float]) -> dict:
    arr = np.array(returns, dtype=float)
    return {"avg_return": float(arr.mean()), "volatility": float(arr.std(ddof=1)), "max_drawdown": max_drawdown(equity)}
```

#### 5. Hard - Quant Report Tool

题目：把指标和解释 prompt 打包给 LLM/Agent。

讲解：计算由 Python 完成，解释交给 LLM 审查。

```python
def quant_report_tool(symbol: str, prices: list[float]) -> dict:
    rets = returns_from_prices(prices)
    equity = [1.0]
    for r in rets:
        equity.append(equity[-1] * (1 + r))
    metrics = strategy_metrics(rets, equity)
    prompt = f"请审查 {symbol} 的策略指标: {metrics}"
    return {"ok": True, "data": {"metrics": metrics, "prompt": prompt}}
```
