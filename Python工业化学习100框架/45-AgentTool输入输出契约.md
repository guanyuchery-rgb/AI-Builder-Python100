## 今日最小可运行任务

```python
def tool_result(ok: bool, data=None, error: str | None = None) -> dict:
    return {"ok": ok, "data": data, "error": error}

def double_tool(payload: dict) -> dict:
    if "value" not in payload:
        return tool_result(False, error="value is required")
    return tool_result(True, data={"double": payload["value"] * 2})

print(double_tool({"value": 3}))
```
