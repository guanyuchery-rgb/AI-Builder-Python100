### 5 道基础巩固题

#### 1. 基础巩固 - 函数作为值

题目：把函数赋给变量再调用。

讲解：理解装饰器前置知识。

```python
def say_hi():
    print("hi")

fn = say_hi
fn()
```

#### 2. 基础巩固 - 最小装饰器

题目：给函数加开始/结束。

讲解：先不带参数。

```python
def log_call(func):
    def wrapper():
        print("开始")
        func()
        print("结束")
    return wrapper

def task():
    print("运行任务")

logged_task = log_call(task)
logged_task()
```

#### 3. 基础巩固 - 语法糖

题目：使用 @ 装饰器。

讲解：和手动包装等价。

```python
@log_call
def task2():
    print("运行任务2")

task2()
```

#### 4. 基础巩固 - with 读取文件

题目：用 with 管理文件。

讲解：资源用完自动关闭。

```python
with open("report.md", "w", encoding="utf-8") as file:
    file.write("hello")
```

#### 5. 基础巩固 - 计时雏形

题目：打印任务前后，不计算真实时间。

讲解：下一步再加 time。

```python
def simple_timer(func):
    def wrapper():
        print("timer start")
        func()
        print("timer end")
    return wrapper
```
