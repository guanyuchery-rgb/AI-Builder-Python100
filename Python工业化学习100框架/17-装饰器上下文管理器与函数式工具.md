## 知识点 1：装饰器 Decorator

### 定义

装饰器是包装函数的函数，用来在不改动原函数主体的情况下增加行为。

### 为什么存在

工程里常需要统一增加日志、计时、权限、缓存、重试。装饰器能减少重复代码。

### 最小案例

```python
from functools import wraps
from time import perf_counter


def timer(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        start = perf_counter()
        result = func(*args, **kwargs)
        elapsed = perf_counter() - start
        print(f"{func.__name__} 用时: {elapsed:.4f}s")
        return result
    return wrapper


@timer
def calculate_sum(numbers: list[int]) -> int:
    return sum(numbers)


print(calculate_sum(list(range(10000))))
```

### 常见错误

- 忘记返回 `result`。
- 忘记使用 `functools.wraps`。
- 装饰器里吞掉异常。

### 工程应用

- 记录函数耗时。
- API 调用重试。
- Agent tool 日志。
- 缓存计算结果。

### 未来扩展

- 带参数装饰器。
- `lru_cache`。
- tracing。

## 知识点 2：上下文管理器 Context Manager

### 定义

上下文管理器用 `with` 管理资源的进入和退出，例如文件、连接、锁。

### 为什么存在

资源需要正确关闭。`with` 能保证即使出错，也能执行清理逻辑。

### 最小案例

```python
from pathlib import Path

path = Path("day17.txt")

with path.open("w", encoding="utf-8") as file:
    file.write("context manager example")
```

### 常见错误

- 手动 open 后忘记 close。
- 在 `with` 外继续使用已关闭文件。
- 不理解异常发生时也会退出上下文。

### 工程应用

- 文件读写。
- 数据库连接。
- 网络连接。
- 临时资源管理。

### 未来扩展

- `contextlib.contextmanager`。
- async context manager。
- transaction。

## Debug 日志

- 文件为空：检查是否写入后关闭或 flush。
- 装饰器后函数名变成 wrapper：使用 `@wraps`。
- 日志没打印：确认装饰器是否真的加在函数上。

### 5 道递进题

#### 1. Easy - timer decorator

题目：打印函数耗时。

讲解：装饰器适合给函数加计时、日志、缓存。

```python
import time
from functools import wraps

def timer(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        start = time.perf_counter()
        result = func(*args, **kwargs)
        print(f"{func.__name__}: {time.perf_counter() - start:.4f}s")
        return result
    return wrapper
```

#### 2. Easy - retry decorator

题目：失败时重试指定次数。

讲解：API client 和 Agent tool 都需要可复用 retry。

```python
from functools import wraps

def retry(attempts: int = 3):
    def decorate(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            last_error = None
            for _ in range(attempts):
                try:
                    return func(*args, **kwargs)
                except Exception as error:
                    last_error = error
            raise RuntimeError("retry failed") from last_error
        return wrapper
    return decorate
```

#### 3. Medium - contextmanager

题目：进入时打印 start，退出时打印 done。

讲解：上下文管理器适合管理资源生命周期。

```python
from contextlib import contextmanager

@contextmanager
def task_scope(name: str):
    print(f"start {name}")
    try:
        yield
    finally:
        print(f"done {name}")
```

#### 4. Medium - partial 固定参数

题目：固定一个汇率参数，生成美元转人民币函数。

讲解：函数式工具能减少重复参数传递。

```python
from functools import partial

def convert(amount: float, rate: float) -> float:
    return amount * rate

usd_to_cny = partial(convert, rate=7.2)
```

#### 5. Hard - 简单缓存装饰器

题目：缓存函数结果，避免重复请求。

讲解：数据获取、Embedding、LLM 调用都常用缓存降低成本。

```python
from functools import wraps

def simple_cache(func):
    cache = {}
    @wraps(func)
    def wrapper(key):
        if key not in cache:
            cache[key] = func(key)
        return cache[key]
    return wrapper
```
