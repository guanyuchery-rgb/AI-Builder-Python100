### 定义

重复执行相似动作。

### 为什么存在

它让当前阶段的代码更容易看懂、运行和复查。
## 知识点 2：累计 Accumulator

### 定义

用一个变量保存逐步累加的结果。

### 为什么存在

它让当前阶段的代码更容易看懂、运行和复查。

### 5 道基础巩固题

#### 1. 基础巩固 - 打印序号

题目：打印 Day1 到 Day5。

讲解：先熟悉 range 起止。

```python
for day in range(1, 6):
    print(f"Day{day}")
```

#### 2. 基础巩固 - 累计求和

题目：计算 1 到 100 的和。

讲解：累计变量从 0 开始。

```python
total = 0
for n in range(1, 101):
    total = total + n
print(total)
```

#### 3. 基础巩固 - 统计偶数

题目：统计 1 到 20 里有几个偶数。

讲解：循环里可以放 if。

```python
count = 0
for n in range(1, 21):
    if n % 2 == 0:
        count = count + 1
print(count)
```

#### 4. 基础巩固 - 模拟每日学习

题目：连续 7 天，每天增加 2 小时。

讲解：这是最小自动化思维。

```python
hours = 0
for day in range(1, 8):
    hours = hours + 2
    print(f"第 {day} 天，累计 {hours} 小时")
```

#### 5. 基础巩固 - 收益累计简化版

题目：给定每天固定收益 1%，循环 5 天。

讲解：先不用列表，理解重复计算。

```python
money = 1000.0
for day in range(1, 6):
    money = money * 1.01
    print(f"第 {day} 天：{money:.2f}")
```
