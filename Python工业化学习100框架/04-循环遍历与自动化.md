## 知识点 1：for 循环

### 定义

`for` 循环会对可迭代对象中的每个元素重复执行代码。

### 为什么存在

真实工作经常要处理很多值：行、文件、股票代码、用户、任务、消息。

### 最小案例

```python
symbols = ["AAPL", "MSFT", "NVDA"]

for symbol in symbols:
    print(f"准备下载 {symbol} 的数据")
```

### 常见错误

- 遍历列表时同时修改列表。
- 明明可以直接遍历，却写复杂下标。
- 忘记 `range(5)` 到 4 结束。

### 工程应用

- 遍历文件夹。
- 遍历股票代码。
- 遍历 API 返回结果。
- 遍历 Agent 任务列表。

### 未来扩展

- 列表推导式。
- 生成器。
- 异步遍历。
- 批处理任务。

## 知识点 2：while 循环

### 定义

`while` 循环会在条件为真时持续执行。

### 为什么存在

有些任务不知道会重复几次，例如重试、等待任务完成、读取用户输入、处理队列。

### 最小案例

```python
attempts = 0

while attempts < 3:
    print(f"第 {attempts + 1} 次尝试")
    attempts += 1
```

### 常见错误

- 条件永远不变，导致死循环。
- 少一次或多一次的边界错误。
- 循环体里塞太多逻辑。

### 工程应用

- 请求失败后重试。
- 轮询任务状态。
- 直到输入合法为止。
- 处理任务队列。

### 未来扩展

- retry 机制。
- queue。
- scheduler。
- 后台任务。

## 现代代码示例：批量计算收益率

```python
prices = [100, 102, 101, 105]

returns = []
for index in range(1, len(prices)):
    previous_price = prices[index - 1]
    current_price = prices[index]
    daily_return = (current_price - previous_price) / previous_price
    returns.append(daily_return)

for index, rate in enumerate(returns, start=1):
    print(f"第 {index} 天收益率: {rate:.2%}")
```

### 5 道递进题

#### 1. Easy - Running Sum

关联题：[LeetCode 1480](https://leetcode.com/problems/running-sum-of-1d-array/)

题目：返回数组前缀和。

讲解：前缀和是很多数据指标的基础。

```python
# 输入: [1,2,3]
# 输出: [1,3,6]
def running_sum(nums: list[int]) -> list[int]:
    total = 0
    ans = []
    for n in nums:
        total += n
        ans.append(total)
    return ans
```

#### 2. Easy - Two Sum

关联题：[LeetCode 1](https://leetcode.com/problems/two-sum/)

题目：找两个数下标，使它们和等于 target。

讲解：用 dict 记录“之前见过的数”，避免双重循环。

```python
# seen[value] = index
def two_sum(nums: list[int], target: int) -> list[int]:
    seen = {}
    for i, n in enumerate(nums):
        need = target - n
        if need in seen:
            return [seen[need], i]
        seen[n] = i
    return []
```

#### 3. Medium - 股票最大利润

关联题：[LeetCode 121](https://leetcode.com/problems/best-time-to-buy-and-sell-stock/)

题目：只能买卖一次，求最大利润。

讲解：遍历时维护历史最低买入价。

```python
# 输入: 每天价格
# 输出: 最大利润，不能赚钱则 0
def max_profit(prices: list[float]) -> float:
    min_price = float("inf")
    best = 0.0
    for price in prices:
        min_price = min(min_price, price)
        best = max(best, price - min_price)
    return best
```

#### 4. Medium - 移动平均

题目：给价格序列和窗口，返回每个窗口平均值。

讲解：这是量化指标里最常见的滚动计算。

```python
# 简单版本: 每次切片求和，先保证正确
# 以后再优化成滑动窗口
def moving_average(values: list[float], window: int) -> list[float]:
    if window <= 0:
        raise ValueError("window must be positive")
    ans = []
    for i in range(window, len(values) + 1):
        ans.append(sum(values[i-window:i]) / window)
    return ans
```

#### 5. Hard - 日志异常扫描

题目：给日志行列表，返回包含 `ERROR` 或 `timeout` 的行号和内容。

讲解：自动化脚本常做的事就是遍历、筛选、定位问题。

```python
# 输出: [(行号, 内容)]，行号从 1 开始更符合人类阅读
def scan_errors(lines: list[str]) -> list[tuple[int, str]]:
    hits = []
    for line_no, line in enumerate(lines, start=1):
        text = line.lower()
        if "error" in text or "timeout" in text:
            hits.append((line_no, line))
    return hits
```
