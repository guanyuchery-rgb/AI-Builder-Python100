## 知识点 1：推导式 Comprehension

### 定义

推导式是一种从已有序列生成新序列的简洁写法，常见形式包括 list comprehension、dict comprehension、set comprehension。

### 为什么存在

数据处理经常需要过滤、转换、映射。推导式能让简单转换更清楚。

### 最小案例

```python
prices = [100, 102, 101, 105]
returns = [
    (prices[index] - prices[index - 1]) / prices[index - 1]
    for index in range(1, len(prices))
]

print([f"{rate:.2%}" for rate in returns])
```

### 常见错误

- 为了简短写得太复杂。
- 在推导式里塞入太多条件。
- 忘记可读性比炫技重要。

### 工程应用

- 数据字段清洗。
- 批量转换 API 结果。
- 构造特征列表。
- LLM chunk 元数据生成。

### 未来扩展

- pandas 向量化。
- generator pipeline。
- map/filter。

## 知识点 2：生成器 Generator

### 定义

生成器是一种按需产出数据的对象，使用 `yield` 创建。

### 为什么存在

当数据很大时，不应该一次性全部加载到内存。生成器能按需处理。

### 最小案例

```python
def moving_window(values: list[float], window_size: int):
    for index in range(len(values) - window_size + 1):
        yield values[index:index + window_size]


prices = [100, 102, 101, 105, 107]
for window in moving_window(prices, 3):
    print(window)
```

### 常见错误

- 生成器只能迭代一次。
- 调试时看不到完整内容，需要 `list(generator)`。
- 不适合需要反复随机访问的场景。

### 工程应用

- 大文件逐行处理。
- 时间序列滑动窗口。
- LLM 文档切分。
- Agent 任务流。

### 未来扩展

- itertools。
- streaming data。
- async generator。

## Debug 日志

- 结果为空：生成器已经被消费过。
- 代码难读：把复杂推导式改回普通 for 循环。
- 内存变高：检查是否一次性 `list()` 了大数据。

### 5 道递进题

#### 1. Easy - 推导式平方

题目：返回所有正数的平方。

讲解：推导式适合短小、单一规则的数据转换。

```python
def square_positive(nums: list[int]) -> list[int]:
    return [n * n for n in nums if n > 0]
```

#### 2. Easy - 价格生成器

题目：只产出合法价格。

讲解：生成器适合流式数据，不一次性占满内存。

```python
from collections.abc import Iterator

def valid_prices(values: list[float]) -> Iterator[float]:
    for value in values:
        if value > 0:
            yield value
```

#### 3. Medium - 截取前 N 个

题目：从任意 iterator 中取前 n 个元素。

讲解：流数据处理要能控制消费量。

```python
from itertools import islice
from collections.abc import Iterable

def take(iterable: Iterable, n: int) -> list:
    # islice 不会消费超过 n 个元素
    return list(islice(iterable, n))
```

#### 4. Medium - 扁平化嵌套列表

关联题：[LeetCode 341](https://leetcode.com/problems/flatten-nested-list-iterator/)

题目：把嵌套 list 扁平化。

讲解：递归生成器让结构转换更干净。

```python
def flatten(items):
    for item in items:
        if isinstance(item, list):
            yield from flatten(item)
        else:
            yield item
```

#### 5. Hard - 流式最大回撤

题目：一边接收净值，一边产出当前最大回撤。

讲解：实时 Quant 指标不能总是等全量数据。

```python
from collections.abc import Iterator

def drawdown_stream(equity: list[float]) -> Iterator[float]:
    peak = None
    worst = 0.0
    for value in equity:
        peak = value if peak is None else max(peak, value)
        worst = min(worst, value / peak - 1)
        yield worst
```
