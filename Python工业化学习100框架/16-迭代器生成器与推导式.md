### 5 道基础巩固题

#### 1. 基础巩固 - 平方列表

题目：用推导式生成平方。

讲解：推导式先写简单。

```python
squares = [n * n for n in range(1, 6)]
print(squares)
```

#### 2. 基础巩固 - 筛选正数

题目：只保留正收益。

讲解：推导式可以加 if。

```python
returns = [0.01, -0.02, 0.03]
positive = [r for r in returns if r > 0]
print(positive)
```

#### 3. 基础巩固 - 字符串清洗列表

题目：批量清洗 symbol。

讲解：复用 Day07。

```python
symbols = [" aapl ", " msft "]
clean = [s.strip().upper() for s in symbols]
print(clean)
```

#### 4. 基础巩固 - 最小生成器

题目：yield 逐个产出数字。

讲解：先理解逐个产出。

```python
def numbers():
    yield 1
    yield 2
    yield 3

for n in numbers():
    print(n)
```

#### 5. 基础巩固 - 价格生成器

题目：只产出有效价格。

讲解：生成器适合流式处理。

```python
def valid_prices(prices):
    for price in prices:
        if price > 0:
            yield price

for price in valid_prices([100, -1, 102]):
    print(price)
```
