### 最小案例

```python
def normalize_tag(text):
    return text.strip().lower().replace(" ", "-")

record = {"topic": " Python Basics ", "hours": 2}
tags = {normalize_tag(record["topic"]), "python"}
print(record)
print(tags)
```

### 常见错误

- 字典 key 拼写不一致，导致取值失败。
- 字符串没有先 `strip()`，比较时看起来一样但实际不同。
- 集合无序，不要依赖 set 的输出顺序。

## 基础详细讲解

### 字符串、字典与集合

字符串处理文本，字典处理 key-value 映射，集合处理去重和成员判断。三者是实际项目中最常见的数据结构：文件路径、字段名、配置项、用户输入、模型输出，很多都是文本和映射。

### 手写顺序

- 字符串先练 `strip()`、`lower()`、`split()`，它们能解决大量清洗问题。
- 字典先练通过 key 取值和更新，例如 `record["score"]`。
- 集合先练去重和判断是否出现过，例如 `x in seen`。

### 今天必须真正弄懂

- **输入**：今天这段代码从哪里拿到数据，是手写值、终端输入、文件，还是函数参数。
- **处理**：每一行代码改变了什么状态，例如变量变了、列表多了一个元素、判断走了某个分支。
- **输出**：结果给谁看，是终端输出、返回值、文件，还是后面函数继续使用。

## 5 道基础巩固题

#### 1. 清洗主题文本

题目：去掉两边空格并转小写。

讲解：文本清洗第一步。

```python
text = " Python Basics "
clean = text.strip().lower()
print(clean)
```

#### 2. 字典保存记录

题目：用 dict 保存一条学习记录。

讲解：字典表达键值数据。

```python
record = {"topic": "python", "hours": 2}
print(record["topic"])
```

#### 3. 安全取值

题目：用 get 读取可能不存在的字段。

讲解：避免 KeyError。

```python
record = {"topic": "python"}
print(record.get("hours", 0))
```

#### 4. 标签去重

题目：用 set 去掉重复标签。

讲解：搜索和分类常用。

```python
tags = {"python", "data", "python"}
print(tags)
```

#### 5. 标签标准化函数

题目：把文本变成统一标签。

讲解：为后面数据清洗做准备。

```python
def normalize_tag(text):
    return text.strip().lower().replace(" ", "-")

print(normalize_tag(" Python Basics "))
```
