## 基础详细讲解

### 变量、类型转换与输入输出

变量可以理解成给数据贴标签。`input()` 得到的一定是字符串，哪怕用户输入的是 `18`，它也先是文字；要参与计算，就必须用 `int()` 或 `float()` 转换。零基础阶段最重要的是分清“看起来像数字”和“真正能计算的数字”。

### 手写顺序

- 变量名写清含义，例如 `height_cm` 比 `h` 更适合复习。
- 输入后立刻转换，并用一次 `print(type(value))` 检查类型。
- 用 f-string 把说明文字和结果合在一起，例如 `print(f"BMI={bmi}")`。

### 今天必须真正弄懂

- **输入**：今天这段代码从哪里拿到数据，是手写值、终端输入、文件，还是函数参数。
- **处理**：每一行代码改变了什么状态，例如变量变了、列表多了一个元素、判断走了某个分支。
- **输出**：结果给谁看，是终端输出、返回值、文件，还是后面函数继续使用。

### 和后续知识的关系

数据分析和 Quant 里，很多错误来自字段类型不对。今天练的是把外部输入变成可计算数据。

### 零基础拆解：变量、类型和输入转换

#### 这一天到底在练什么

`input()` 拿到的是 text，不是 number。统计和数据分析里很多 bug 都来自类型没转好，所以今天要反复练：输入 -> 转换 -> 计算 -> 输出。

#### 手写时按这个顺序

- 先用 `name = input("name: ")` 保存文本。
- 再用 `hours = float(input("hours: "))` 保存可计算数字。
- 最后用 f-string 输出一句完整说明。

#### 最常见卡点

如果遇到 `TypeError`，通常是字符串和数字混在一起算；如果遇到 `ValueError`，通常是输入内容无法转成数字。

## 5 道基础巩固题

#### 1. 变量保存信息

题目：用变量保存主题和分钟数并打印。

讲解：变量是给数据起名字。

```python
topic = "Python"
minutes = 120
print(topic)
print(minutes)
```

#### 2. 输入姓名

题目：读取一个昵称并输出欢迎语。

讲解：只做输入输出，不做判断。

```python
name = input("昵称：")
print(f"欢迎，{name}")
```

#### 3. 小时转分钟

题目：读取小时数并转换成分钟。

讲解：练习 float 和乘法。

```python
hours = float(input("小时数："))
minutes = hours * 60
print(f"分钟数：{minutes}")
```

#### 4. 收益率格式化

题目：把小数收益率格式化成百分比。

讲解：Quant 中常见格式。

```python
return_rate = 0.0123
print(f"收益率：{return_rate:.2%}")
```

#### 5. 类型观察

题目：打印 `type()` 结果。

讲解：理解输入默认是 str。

```python
text = input("输入任意内容：")
print(type(text))
```
