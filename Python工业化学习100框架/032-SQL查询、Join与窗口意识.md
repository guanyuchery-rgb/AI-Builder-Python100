### 最小案例

```python
import sqlite3

with sqlite3.connect("research.db") as conn:
    rows = conn.execute("""
        SELECT symbol, AVG(avg_return) AS mean_return
        FROM metrics
        GROUP BY symbol
        ORDER BY mean_return DESC
    """).fetchall()
print(rows)
```
