```python
def max_profit(prices):
    min_price = float("inf")
    best = 0
    for price in prices:
        min_price = min(min_price, price)
        best = max(best, price - min_price)
    return best
```

3. **数据题**：准备 5-10 条虚拟样例数据，覆盖正常值、空值、重复值和极端值。
4. **函数题**：把核心逻辑封装成 1 个函数，函数名、参数名和返回值必须能读懂。
5. **测试题**：写 2 个正常测试和 1 个边界测试，确认修改后行为不变。
6. **算法题**：完成 [Valid Palindrome](https://leetcode.com/problems/valid-palindrome/)，题型：双指针。提示：从两端收缩，训练边界移动和字符过滤。
7. **项目题**：把今天结果接入 `本地知识助手` 的一个小模块，并在 README 里写 3 行使用说明。

### 难度控制

Medium 为主，逐步接触图、堆、检索和结构化解析。 把题目理解成 LLM/Agent 的工具能力：输入契约、输出结构、可验证结果。 如果算法题暂时写不出完整代码，先写暴力思路、输入输出样例和复杂度，再补优化版。
