### 5 道基础巩固题

#### 1. 基础巩固 - 学习清单

题目：保存 3 个主题并逐行打印。

讲解：列表让多个值放在一起。

```python
topics = ["变量", "分支", "循环"]
for topic in topics:
    print(topic)
```

#### 2. 基础巩固 - 价格平均值

题目：用列表保存价格并计算平均值。

讲解：先用 sum 和 len。

```python
prices = [100.0, 102.0, 101.0]
average = sum(prices) / len(prices)
print(f"平均价格：{average:.2f}")
```

#### 3. 基础巩固 - 找最大值

题目：找出最高价格。

讲解：先用内置 max，后面再手写算法。

```python
prices = [100, 105, 99, 108]
print(max(prices))
```

#### 4. 基础巩固 - 手写最大值

题目：不用 max，手写循环找最大值。

讲解：训练算法步骤。

```python
prices = [100, 105, 99, 108]
best = prices[0]
for price in prices:
    if price > best:
        best = price
print(best)
```

#### 5. 基础巩固 - 统计正数

题目：统计收益率列表里正数个数。

讲解：循环 + if + 计数器。

```python
returns = [0.01, -0.02, 0.03, 0.00]
count = 0
for r in returns:
    if r > 0:
        count = count + 1
print(count)
```
