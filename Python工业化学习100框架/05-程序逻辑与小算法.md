## 知识点 1：问题分解

### 定义

问题分解是把任务拆成输入、处理过程和输出。

### 为什么存在

分解清楚后，代码更容易测试、调试、复用，也更容易让 AI 帮你审查。

### 最小案例

```python
def is_even(number: int) -> bool:
    return number % 2 == 0

print(is_even(10))
print(is_even(7))
```

### 常见错误

- 输入、计算、输出混在一起。
- 函数只打印不返回。
- 不测边界值。

### 工程应用

- 数据验证。
- 指标计算。
- 业务规则。
- Agent 工具函数。

### 未来扩展

## 知识点 2：测试思维

### 定义

测试思维是用具体例子证明函数在正常、边界、异常情况下是否符合预期。

### 为什么存在

没有测试，只能说明代码某一次碰巧跑通，不能说明它可靠。

### 最小案例

```python
def simple_return(start_price: float, end_price: float) -> float:
    return (end_price - start_price) / start_price

assert simple_return(100, 110) == 0.10
assert simple_return(100, 90) == -0.10
```

### 常见错误

- 用 `assert` 比较浮点数时忽略精度问题。
- 只测正常情况。
- 忘记起始价格为 0 的无效情况。

### 工程应用

- 验证金融公式。
- 检查清洗逻辑。
- 确认 Agent 工具行为。

### 未来扩展

- `pytest`。
- fixture。
- 属性测试。
- CI 检查。

## 现代代码示例：最大回撤简化版

```python
def max_drawdown(values: list[float]) -> float:
    peak = values[0]
    worst_drawdown = 0.0

    for value in values:
        if value > peak:
            peak = value
        drawdown = (value - peak) / peak
        if drawdown < worst_drawdown:
            worst_drawdown = drawdown

    return worst_drawdown

portfolio_values = [100, 120, 110, 130, 90, 95]
print(f"最大回撤: {max_drawdown(portfolio_values):.2%}")
```

### 5 道递进题

#### 1. Easy - Climbing Stairs

关联题：[LeetCode 70](https://leetcode.com/problems/climbing-stairs/)

题目：每次走 1 或 2 阶，求到第 n 阶的方法数。

讲解：这是最小动态规划，重点是状态转移。

```python
# dp[i] = 到第 i 阶的方法数
# 只保留前两个状态，空间更干净
def climb_stairs(n: int) -> int:
    if n <= 2:
        return n
    a, b = 1, 2
    for _ in range(3, n + 1):
        a, b = b, a + b
    return b
```

#### 2. Medium - Maximum Subarray

关联题：[LeetCode 53](https://leetcode.com/problems/maximum-subarray/)

题目：求连续子数组最大和。

讲解：这像收益序列里找最强连续区间。

```python
# current 表示以当前位置结尾的最佳结果
# best 表示全局最佳结果
def max_subarray(nums: list[int]) -> int:
    current = best = nums[0]
    for n in nums[1:]:
        current = max(n, current + n)
        best = max(best, current)
    return best
```

#### 3. Medium - Merge Sorted Array

关联题：[LeetCode 88](https://leetcode.com/problems/merge-sorted-array/)

题目：合并两个有序数组。

讲解：数据管道里常见“两个已排序来源合并”。

```python
# 返回新列表版本，先掌握双指针思想
def merge_sorted(a: list[int], b: list[int]) -> list[int]:
    i = j = 0
    ans = []
    while i < len(a) and j < len(b):
        if a[i] <= b[j]:
            ans.append(a[i]); i += 1
        else:
            ans.append(b[j]); j += 1
    return ans + a[i:] + b[j:]
```

#### 4. Medium - Top K 高频标签

题目：统计标签出现次数，返回出现最多的 K 个。

讲解：这是日志分析、Prompt 标签统计、交易事件统计的通用题。

```python
from collections import Counter

# Counter 帮你做频次统计，most_common 直接取前 K
def top_k_tags(tags: list[str], k: int) -> list[tuple[str, int]]:
    return Counter(tags).most_common(k)
```

#### 5. Hard - 最大回撤

题目：给净值序列，计算最大回撤。

讲解：Quant 里最大回撤不是最小值，而是历史高点后的最大下跌。

```python
# 输出为负数，例如 -0.25 表示从高点跌了 25%
def max_drawdown(equity: list[float]) -> float:
    peak = equity[0]
    worst = 0.0
    for value in equity:
        peak = max(peak, value)
        drawdown = value / peak - 1
        worst = min(worst, drawdown)
    return worst
```
