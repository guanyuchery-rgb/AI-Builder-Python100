   - 题型：动态规划

   - 解题步骤：先定义 dp 状态，再写 base case，最后写状态转移并用小样例手算一遍。

### 一题多解法拆解：Contains Duplicate

- 解法 2：set 扫描。

推荐先写最容易懂的版本。

### 带注释参考代码：Contains Duplicate

```python
def contains_duplicate(nums):
    seen = set()
    for num in nums:
        if num in seen:
            return True
        seen.add(num)
    return False
```
