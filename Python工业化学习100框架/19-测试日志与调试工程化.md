### 5 道基础巩固题

#### 1. 基础巩固 - assert 测试

题目：测试 add。

讲解：先用 Python 自带 assert。

```python
def add(a, b):
    return a + b

assert add(2, 3) == 5
```

#### 2. 基础巩固 - 收益率测试

题目：测试一个确定案例。

讲解：测试让公式更可靠。

```python
def simple_return(start, end):
    return (end - start) / start

assert simple_return(100, 110) == 0.1
```

#### 3. 基础巩固 - logging info

题目：记录运行过程。

讲解：项目里少用 print。

```python
import logging

logging.basicConfig(level=logging.INFO)
logging.info("job started")
```

#### 4. 基础巩固 - logging error

题目：捕获异常并记录。

讲解：失败要有痕迹。

```python
import logging

try:
    value = int("abc")
except ValueError:
    logging.error("转换失败")
```

#### 5. 基础巩固 - 最小复现

题目：把问题缩到三行。

讲解：Debug 先缩小。

```python
data = {}
print(data.get("price", "N/A"))
```
