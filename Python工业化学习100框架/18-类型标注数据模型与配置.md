### 5 道基础巩固题

#### 1. 基础巩固 - 函数标注

题目：给参数和返回值标注。

讲解：标注帮助阅读，不替代测试。

```python
def add(a: int, b: int) -> int:
    return a + b
```

#### 2. 基础巩固 - 收益率标注

题目：给收益率函数加标注。

讲解：开始形成工程习惯。

```python
def simple_return(start_price: float, end_price: float) -> float:
    return (end_price - start_price) / start_price
```

#### 3. 基础巩固 - 配置 dataclass

题目：集中保存配置。

讲解：避免参数散落。

```python
from dataclasses import dataclass

@dataclass
class Config:
    data_dir: str = "data"
    output_dir: str = "output"
```

#### 4. 基础巩固 - 路径转换

题目：把配置路径变成 Path。

讲解：路径用 Path 更稳定。

```python
from pathlib import Path

config = Config()
data_path = Path(config.data_dir)
print(data_path)
```

#### 5. 基础巩固 - 配置校验

题目：检查目录名是否为空。

讲解：配置也要验证。

```python
def validate_config(config: Config) -> list[str]:
    errors = []
    if config.data_dir == "":
        errors.append("data_dir 不能为空")
    return errors
```
