## 知识点 1：类型标注 Type Hints

### 定义

类型标注用于说明变量、函数参数和返回值的预期类型。

### 为什么存在

类型标注让代码更容易阅读、补全、检查，也让 AI 更容易理解你的意图。

### 最小案例

```python
def simple_return(start_price: float, end_price: float) -> float:
    return (end_price - start_price) / start_price


def normalize_symbol(symbol: str) -> str:
    return symbol.strip().upper()
```

### 常见错误

- 写了类型标注但运行时不验证。
- 标注太复杂影响理解。
- `list`、`dict` 内部类型不明确。

### 工程应用

- 函数边界。
- API 参数。
- Agent tool schema。
- Quant 指标函数。

### 未来扩展

- `mypy`。
- `typing.Protocol`。
- Pydantic。

## 知识点 2：配置 Config

### 定义

配置是程序运行时可调整的参数，例如数据路径、股票代码列表、阈值、模型名。

### 为什么存在

把配置和代码分离，可以避免每次改参数都改程序逻辑。

### 最小案例

```python
from dataclasses import dataclass
from pathlib import Path


@dataclass
class BacktestConfig:
    data_path: Path
    symbols: list[str]
    max_weight: float


config = BacktestConfig(
    data_path=Path("prices.csv"),
    symbols=["AAPL", "MSFT"],
    max_weight=0.2,
)

print(config)
```

### 常见错误

- 把路径、阈值、密钥写死在代码中。
- 配置没有默认值说明。
- 配置改变后结果不可复现。

### 工程应用

- 回测参数。
- LLM 模型选择。
- 数据目录。
- Agent 工具开关。

### 未来扩展

- `.env`。
- YAML/TOML。
- Pydantic Settings。

#### 1. Easy - 类型标注函数

题目：写一个计算收益率的函数。

讲解：类型标注是给未来自己和 AI 更清楚的边界。

```python
def simple_return(start_price: float, end_price: float) -> float:
    if start_price <= 0:
        raise ValueError("start_price must be positive")
    return end_price / start_price - 1
```

#### 2. Easy - Config dataclass

题目：定义项目配置。

讲解：配置集中管理，避免魔法数字散落。

```python
from dataclasses import dataclass

@dataclass
class AppConfig:
    data_dir: str
    default_symbol: str = "SPY"
    timeout: int = 10
```

#### 3. Medium - TypedDict 行情记录

题目：给 quote dict 定义类型形状。

讲解：TypedDict 适合描述来自 JSON/API 的字典结构。

```python
from typing import TypedDict

class Quote(TypedDict):
    symbol: str
    price: float
    volume: int
```

#### 4. Medium - 路径配置解析

题目：把配置里的 data_dir 转成 Path。

讲解：路径统一用 Path，跨项目更稳。

```python
from pathlib import Path

def resolve_data_dir(config: AppConfig) -> Path:
    path = Path(config.data_dir).expanduser()
    path.mkdir(parents=True, exist_ok=True)
    return path
```

#### 5. Hard - Tool 输入校验结果

题目：返回结构化校验结果。

讲解：Agent tool 不只要会跑，还要能解释为什么不能跑。

```python
from dataclasses import dataclass

@dataclass
class ValidationResult:
    ok: bool
    errors: list[str]

def validate_quote_payload(payload: dict) -> ValidationResult:
    errors = []
    if not payload.get("symbol"):
        errors.append("symbol is required")
    if payload.get("price", 0) <= 0:
        errors.append("price must be positive")
    return ValidationResult(ok=len(errors) == 0, errors=errors)
```
