## 今日最小可运行任务

```python
import sqlite3

with sqlite3.connect("research.db") as conn:
    conn.execute("CREATE TABLE IF NOT EXISTS metrics (symbol TEXT, avg_return REAL)")
    conn.execute("INSERT INTO metrics VALUES (?, ?)", ("A", 0.01))
    rows = conn.execute("SELECT symbol, avg_return FROM metrics").fetchall()
print(rows)
```
