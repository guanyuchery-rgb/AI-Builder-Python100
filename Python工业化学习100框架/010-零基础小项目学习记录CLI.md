### 最小案例

def append_log(topic, minutes):

append_log("Python function", 120)

### 基础知识深讲：小项目闭环与状态保存

Day10 的重点是把零散语法连成一个可用系统。一个小项目至少要有输入、处理、输出和保存。比如学习记录 CLI：输入主题和用时，处理成结构化记录，输出确认信息，并保存到文件。这样它才不是临时练习，而是一个可复查工具。

项目里的“状态”指系统当前保存了什么信息。学习记录保存到文件后，下一次运行还能读取，这就是最简单的状态管理。以后数据库、缓存、Agent memory、回测结果表，本质上都是更复杂的状态保存。

小项目不要一开始追求功能多。菜单、界面、统计图、数据库都可以以后加。最小闭环应该先保证一条记录能进入系统、被保存、被读取、被解释。这个顺序能避免你被复杂功能拖住。

README 也是项目的一部分。它回答别人如何运行、输入是什么、输出在哪里、失败时怎么查。对公开作品来说，README 往往是别人理解你工程能力的第一入口。

### 手写顺序

- 先定义一条记录包含哪些字段，例如日期、主题、用时、收获。
- 再写最小功能：新增记录、列出记录、保存到文件。
- 最后补错误处理：空输入、文件不存在、格式不对。

### 今天必须真正弄懂

- **输入**：今天这段代码从哪里拿到数据，是手写值、终端输入、文件，还是函数参数。
- **处理**：每一行代码改变了什么状态，例如变量变了、列表多了一个元素、判断走了某个分支。
- **输出**：结果给谁看，是终端输出、返回值、文件，还是后面函数继续使用。

## 工程化详细讲解

### 如果你完全看不懂，先只抓这一句话

今天这章不是孤立知识点，而是在训练一个真实工程习惯：把「零基础小项目学习记录CLI」放进一个可以运行、可以检查、可以复用的小流程里。

这一阶段的目标是把基础语法变成可复用的小能力。你要从“我会写一段代码”，过渡到“我知道这段代码放在哪里、输入是什么、输出是什么、失败时怎么查”。

### 真实工作流怎么想

你可以按下面这条线理解今天内容：

```text
输入数据 -> 选择数据结构 -> 写成函数 -> 打印/保存结果 -> 记录错误
```

这条线的意思是：先不要追求写得高级。先保证你知道数据从哪里来，经过了哪些变量，最后去了哪里。只要能画出这条数据流，你就已经从“背语法”进入“写程序”。

### 执行过程追踪

学习今天内容时，每写 5 到 10 行代码，就问自己四个问题：

1. 现在有哪些变量？
2. 每个变量是什么类型？
3. 下一行代码会改变哪个变量？
4. 如果输出不对，我应该打印哪里？

这四个问题比多背几个函数更重要。以后做数据分析、Agent tool、量化回测时，真正救命的就是这种追踪能力。

### 最容易混的点

- 不要只看最终答案，要看中间变量怎么变化。
- 不要把函数、循环、文件、列表当成分开的知识，它们最后都会组合在一个任务里。
- 不要一开始就追求“优雅写法”。先写清楚，再减少重复。
- 每个例子都要能说清楚 input、process、output。

## 5 道基础巩固题

#### 1. 追加学习记录

题目：把主题和分钟数写入 CSV。

讲解：完成最小保存闭环。

```python
from pathlib import Path
line = "python,120\n"
Path("learning.csv").open("a", encoding="utf-8").write(line)
```

#### 2. 封装追加函数

题目：把写入逻辑放进函数。

讲解：便于复用。

```python
from pathlib import Path
def append_log(topic, minutes):
    Path("learning.csv").open("a", encoding="utf-8").write(f"{topic},{minutes}\n")
append_log("python", 120)
```

#### 3. 读取记录

题目：读取 learning.csv 并打印。

讲解：确认输出可复查。

```python
from pathlib import Path
print(Path("learning.csv").read_text(encoding="utf-8"))
```

#### 4. 生成 README

题目：写一个最小运行说明。

讲解：开源项目入口。

```python
from pathlib import Path
Path("README.md").write_text("# Learning CLI\n\nRun: python day10_learning_cli.py\n", encoding="utf-8")
```

#### 5. 项目复盘

题目：用三行文字写项目复盘。

讲解：从第一阶段开始沉淀资产。

```python
notes = ["输入：topic/minutes", "处理：append", "输出：learning.csv"]
for note in notes:
    print(note)
```
