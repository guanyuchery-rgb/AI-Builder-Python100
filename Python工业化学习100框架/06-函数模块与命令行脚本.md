## 知识点 1：函数 Function

### 定义

函数把一段逻辑封装成可重复调用的名字。

### 为什么存在

它让当前阶段的代码更容易看懂、运行和复查。
## 知识点 2：返回值 Return

### 定义

函数用 `return` 把结果交回调用者。

### 为什么存在

它让当前阶段的代码更容易看懂、运行和复查。

### 5 道基础巩固题

#### 1. 基础巩固 - 加法函数

题目：写一个最小函数并调用。

讲解：函数从今天才开始出现。

```python
def add(a, b):
    return a + b

print(add(2, 3))
```

#### 2. 基础巩固 - 收益率函数

题目：把 Day02 公式封装成函数。

讲解：公式复用从函数开始。

```python
def simple_return(start_price, end_price):
    return (end_price - start_price) / start_price

print(simple_return(100, 108))
```

#### 3. 基础巩固 - 列表平均值函数

题目：输入列表，返回平均值。

讲解：这里复用 Day05 的列表知识。

```python
def average(values):
    return sum(values) / len(values)

prices = [100, 102, 101]
print(average(prices))
```

#### 4. 基础巩固 - 验证函数

题目：输入价格，返回是否有效。

讲解：函数也可以返回 True/False。

```python
def is_valid_price(price):
    return price > 0

print(is_valid_price(100))
print(is_valid_price(-1))
```

#### 5. 基础巩固 - main 入口

题目：把调用放进 main。

讲解：以后项目都沿用这个结构。

```python
def main():
    price = 100
    end = 108
    print(simple_return(price, end))

if __name__ == "__main__":
    main()
```
