## 基础详细讲解

### 列表、append、遍历与最小算法

列表是有顺序的数据容器。`append()` 的意思是“把一个新元素放到列表末尾”，所以它必须属于某个列表对象，例如 `values.append(3)`。遍历列表时，每次只关注当前元素，然后决定是否更新答案。

### 手写顺序

- 先准备一个小列表，例如 `[3, 8, 2]`，不要一开始就处理复杂数据。
- 找最大值时，把第一个元素当作当前最好答案，再逐个比较和更新。
- 平均值需要两个量：总和 `total` 和数量 `len(values)`，分母不能为 0。

### 今天必须真正弄懂

- **输入**：今天这段代码从哪里拿到数据，是手写值、终端输入、文件，还是函数参数。
- **处理**：每一行代码改变了什么状态，例如变量变了、列表多了一个元素、判断走了某个分支。
- **输出**：结果给谁看，是终端输出、返回值、文件，还是后面函数继续使用。

### 和后续知识的关系

### 零基础拆解：列表遍历和更新答案

#### 这一天到底在练什么

列表保存一组数据，遍历负责一个一个看。最小算法不是背模板，而是维护一个“当前答案”：看到更好的值就更新，否则保持不变。

#### 手写时按这个顺序

- 先准备 `values = [3, 8, 2]`。
- 找最大值时写 `best = values[0]`，不要从 0 猜答案。
- 遍历时只比较 `v` 和 `best`，满足条件才更新。

#### 最常见卡点

`append()` 必须写成 `values.append(x)`，因为它是列表对象的方法。平均值要先确认列表不是空的。

#### 2. 价格平均值

题目：用列表保存价格并计算平均值。

讲解：先用 sum 和 len。

```python
prices = [100.0, 102.0, 101.0]
average = sum(prices) / len(prices)
print(average)
```

#### 3. 找最大值

题目：找出最高价格。

讲解：先用内置 max。

```python
prices = [100, 105, 99, 108]
print(max(prices))
```

#### 4. 手写最大值

题目：不用 max，手写循环找最大值。

讲解：训练算法步骤。

```python
prices = [100, 105, 99, 108]
best = prices[0]
for price in prices:
    if price > best:
        best = price
print(best)
```

#### 5. 统计正收益

题目：统计收益率列表里正数个数。

讲解：循环 + if + 计数器。

```python
returns = [0.01, -0.02, 0.03, 0.00]
count = 0
for r in returns:
    if r > 0:
        count = count + 1
print(count)
```
