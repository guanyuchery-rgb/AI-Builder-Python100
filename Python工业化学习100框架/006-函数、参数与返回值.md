### 定义

把重复逻辑封装成可复用函数，是把一个具体问题拆成可运行、可检查、可复用代码的过程。

### 为什么存在

### 最小案例

```python
def average(values):
    if not values:
        return 0
    return sum(values) / len(values)

print(average([1, 2, 3]))
print(average([]))
```

### 常见错误

- 函数里只 `print` 不 `return`，导致结果不能复用。
- 参数名含义不清楚。
- 把所有代码都塞进一个巨大函数。

## 5 道基础巩固题

#### 1. 加法函数

题目：写一个返回两数之和的函数。

讲解：函数要 return。

```python
def add(a, b):
    return a + b

print(add(2, 3))
```

#### 2. 平均值函数

题目：封装平均值计算。

讲解：处理空列表。

```python
def average(values):
    if not values:
        return 0
    return sum(values) / len(values)

print(average([1, 2, 3]))
```

#### 3. 收益率函数

题目：计算开始价格到结束价格的收益率。

讲解：Quant 常用函数。

```python
def simple_return(start, end):
    return end / start - 1

print(simple_return(100, 105))
```

#### 4. 价格校验函数

题目：判断价格是否大于 0。

讲解：为数据校验做准备。

```python
def is_valid_price(price):
    return price > 0

print(is_valid_price(100))
```

#### 5. main 入口

题目：用 main 管理脚本入口。

讲解：后面 CLI 会用到。

```python
def main():
    print("run")

if __name__ == "__main__":
    main()
```
