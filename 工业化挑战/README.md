# 工业化挑战 IC01-IC50

IC 从 Day15 面向对象与 dataclass 结束后开始。

Day01-Day100 负责知识学习；IC01-IC50 负责工业化项目能力验证。

每个 IC 文件都包含：

- 5 道不重复 LeetCode 官方题，合计 250 道。
- 一个工业化工程任务。
- 数据来源 / 采集 / 清洗 / 分析 / SQL / 可视化 / GitHub 提交要求。
- Review 和求职表达。

## 四个求职项目方向

- Project01：数据分析求职项目（IC01-IC15）
- Project02：Quant 研究求职项目（IC16-IC28）
- Project03：LLM / RAG 求职项目（IC29-IC40）
- Project04：Agent 工作流求职项目（IC41-IC50）
