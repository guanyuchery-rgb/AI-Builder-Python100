## 已吸收的旧 Day 材料

旧 `Day11 - CSV、JSON 与结构化数据` 不再单独放在 migrated 目录里，而是并入 Project01 的早期工业步骤：

- `IC02 - 数据下载与目录设计`：承接 `open()`、路径、原始数据保存、结构化文件输入来源。
- `IC03 - CSVJSON清洗`：承接 CSV/JSON 的读取、字段命名、类型转换、缺失和重复检查。
- `IC04 - Pandas特征表`：承接“结构化记录 -> 表格特征”的升级。

学习时不需要回旧长文。先按 IC02 建目录和数据来源，再按 IC03 读写 CSV/JSON，最后用 IC04 把结构化数据转成可分析特征表。

## 项目完成标准

- 有可运行代码和小样本数据。
- README 写清问题、数据、运行命令、结果和限制。
- 有截图、指标表或报告。
- GitHub 提交历史能看出迭代过程。
- 有一段可以放进简历的项目描述。
