# IC100 后的四个项目沉淀

当前推荐主线见：[../IC100-能力体系](../IC100-能力体系/README.md)。

IC100 不是 100 个散乱练习，而是持续沉淀 4 个可展示项目。

## Project01 Data Analysis

来源：IC020-IC039

最终项目：电商用户行为分析

技术：Python + CSV/JSON + SQL + SQLite + Pandas + NumPy + Matplotlib + Streamlit

交付物：GitHub 仓库、README、示例数据、运行说明、分析报告、Dashboard 截图、简历描述。

## Project02 Quant Research

来源：IC040-IC059

最终项目：Mini Quant Research Framework

技术：Return + Volatility + Sharpe + Drawdown + Slippage + Position + Factor + Backtest

交付物：GitHub 仓库、README、回测结果截图、风险指标说明、策略限制说明、复现脚本、简历描述。

## Project03 LLM Application

来源：IC060-IC079

最终项目：RAG Knowledge Assistant

技术：Prompt + Context + Embedding + Retrieval + RAG + Evaluation + Error Cases

交付物：GitHub 仓库、README、RAG 检索截图、引用回答、失败案例、评估说明、简历描述。

## Project04 Agent Workflow

来源：IC080-IC100

最终项目：Agent Workflow System

技术：Tool + Schema + Memory + Workflow + Planning + Human Review + Observability

交付物：GitHub 仓库、README、Agent 工作流截图、Tool schema、trace 日志、错误恢复说明、简历描述。
