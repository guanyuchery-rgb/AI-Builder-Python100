# LeetCode（Hot Code）300

这是从 IC 中抽离出来的独立算法模块。IC 以后只负责工程项目，LeetCode 单独负责算法刷题记录。

## 学习规则

- 总计 300 道官方 LeetCode。
- 60 天完成，每天 5 道。
- 每道题都保留官方提交记录，方便以后求职展示刷题轨迹。
- 每天输出 `notes.md` 或当天 Markdown 里的复盘：思路、复杂度、错因、二刷标记。
- 不追求一天写很多题，优先保证每题能讲清楚。

## 数量结构

- Easy: 90
- Medium: 210
- Hard: 0

## 每天固定流程

```text
1. 先看 5 题所属类型，不直接写代码。
2. 每题写一句“输入 -> 输出”。
3. 写第一版解法，先通过。
4. 补复杂度和易错点。
5. 在 LeetCode 官方页面提交，留下记录。
```
