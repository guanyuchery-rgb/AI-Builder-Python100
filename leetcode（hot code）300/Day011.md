## 五题清单

### 1. [Move Zeroes](https://leetcode.com/problems/move-zeroes/)

- Slug: `move-zeroes`
- 难度：Easy
- 类型：双指针 / 数组
- 题目解析：双指针适合有序数组、左右边界、原地修改和区间收缩。
- 解题步骤：先确定左右指针含义；每轮只移动能排除答案的一侧；维护结果或写入位置。
- 易错点：移动条件写反，或者覆盖数组后忘记返回有效长度。
- 复杂度记录：写完后补 `Time O(...) / Space O(...)`。

### 2. [Word Pattern](https://leetcode.com/problems/word-pattern/)

- Slug: `word-pattern`
- 难度：Easy
- 类型：哈希表 / 计数
- 题目解析：哈希表把“找过去出现过什么”变成 O(1) 查询。
- 解题步骤：先定义 key 和 value；遍历时先查旧状态，再更新当前状态；最后处理重复值和不存在情况。
- 易错点：把当前元素过早写入哈希表，导致自己和自己匹配。

- Slug: `range-sum-query-immutable`

- 易错点：dp 含义变了但转移没变，或者初始化漏掉空状态。
