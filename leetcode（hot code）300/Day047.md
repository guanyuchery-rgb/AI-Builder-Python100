## 五题清单

### 1. [Palindrome Partitioning](https://leetcode.com/problems/palindrome-partitioning/)

- Slug: `palindrome-partitioning`
- 难度：Medium
- 类型：字符串 / 滑动窗口
- 题目解析：字符串题核心是窗口、计数和边界。
- 解题步骤：右指针扩张窗口；违反条件时左指针收缩；每轮更新最优答案。
- 易错点：窗口收缩后没有同步更新计数，导致重复字符或长度错误。
- 复杂度记录：写完后补 `Time O(...) / Space O(...)`。

### 2. [Triangle](https://leetcode.com/problems/triangle/)

- Slug: `triangle`
- 难度：Medium
- 类型：动态规划 / 前缀和
- 题目解析：DP 题先定义状态，不要先写循环。
- 解题步骤：定义 dp[i] 含义；写转移；初始化边界；按依赖顺序遍历。
- 易错点：dp 含义变了但转移没变，或者初始化漏掉空状态。
- 复杂度记录：写完后补 `Time O(...) / Space O(...)`。

### 4. [Insertion Sort List](https://leetcode.com/problems/insertion-sort-list/)

- Slug: `insertion-sort-list`
- 难度：Medium
- 类型：链表
- 题目解析：链表题先画指针，不要急着写代码。
- 解题步骤：设置 dummy；明确 prev/current/next；每次修改指针后确认链不断。
- 易错点：修改 next 的顺序错，导致链表断掉或成环。
- 复杂度记录：写完后补 `Time O(...) / Space O(...)`。

- 题目解析：树题通常是递归定义问题：当前节点做什么，左右子树返回什么。
- 解题步骤：先写 base case；再处理左右子树；最后合并当前节点结果。
