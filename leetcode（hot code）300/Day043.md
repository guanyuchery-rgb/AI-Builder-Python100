### 1. [Check If There Is A Valid Path In A Grid](https://leetcode.com/problems/check-if-there-is-a-valid-path-in-a-grid/)

- Slug: `check-if-there-is-a-valid-path-in-a-grid`
- 难度：Medium
- 类型：图 / BFS / DFS
- 题目解析：图题重点是状态、邻居、visited 和搜索顺序。
