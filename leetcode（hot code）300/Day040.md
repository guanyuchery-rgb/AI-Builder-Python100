- Slug: `interval-list-intersections`
- 难度：Medium

- 题目解析：链表题先画指针，不要急着写代码。
- 解题步骤：设置 dummy；明确 prev/current/next；每次修改指针后确认链不断。
- 易错点：修改 next 的顺序错，导致链表断掉或成环。
