### 1. [Ransom Note](https://leetcode.com/problems/ransom-note/)

- Slug: `ransom-note`
- 难度：Easy
- 类型：哈希表 / 计数
- 题目解析：哈希表把“找过去出现过什么”变成 O(1) 查询。
- 解题步骤：先定义 key 和 value；遍历时先查旧状态，再更新当前状态；最后处理重复值和不存在情况。
- 易错点：把当前元素过早写入哈希表，导致自己和自己匹配。
