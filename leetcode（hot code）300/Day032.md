### 1. [House Robber](https://leetcode.com/problems/house-robber/)

- Slug: `house-robber`
- 难度：Medium
- 类型：动态规划 / 前缀和
- 题目解析：DP 题先定义状态，不要先写循环。
- 解题步骤：定义 dp[i] 含义；写转移；初始化边界；按依赖顺序遍历。
- 易错点：dp 含义变了但转移没变，或者初始化漏掉空状态。
- 复杂度记录：写完后补 `Time O(...) / Space O(...)`。

- Slug: `binary-tree-right-side-view`

- 类型：树 / DFS / BFS
- 题目解析：树题通常是递归定义问题：当前节点做什么，左右子树返回什么。
- 解题步骤：先写 base case；再处理左右子树；最后合并当前节点结果。
- 易错点：递归返回值含义不清，导致路径、深度、校验混在一起。
